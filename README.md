# SpendWise - Personal Expense Tracker App

Welcome to the documentation for **SpendWise**, a personal finance management application built with **React Native Expo**. SpendWise helps users track their income and expenses, visualize their spending habits with interactive charts, and manage their budget effectively. The app leverages **Supabase** for backend services (authentication and database) and **Redux** for state management.

## Features

### Mobile App (Android/iOS):

1. **User Authentication**:
   - Secure login and registration powered by **Supabase Auth**.
   - Email/password authentication flow.

2. **Expense & Income Tracking**:
   - Easily add, edit, and delete transactions.
   - Categorize spending (Food, Transport, Bills, etc.).
   - Date picker integration for accurate record-keeping.

3. **Interactive Visualizations**:
   - Insightful charts and graphs using `react-native-gifted-charts`.
   - Visual breakdown of expenses by category.

4. **State Management**:
   - Robust data handling with **Redux Toolkit**.
   - Persistent local storage using `AsyncStorage` and `redux-persist`.

5. **Navigation**:
   - Seamless tab-based and stack navigation using **Expo Router**.
   - Smooth transitions between dashboard, transactions, and settings screens.

6. **Responsive Design**:
   - Adaptive UI for various screen sizes using `react-native-responsive-screen`.
   - Customized components for a polished look.

### Tech Stack

- **Frontend**: React Native, Expo, React Redux
- **Backend / Database**: Supabase (PostgreSQL)
- **Navigation**: Expo Router
- **UI Components**: React Native Paper, Vector Icons
- **Charts**: react-native-gifted-charts
